This repository is for demonstration / academic purposes.

Get everything started by running:

```bash
docker-compose up -d
```

Or for frontend dev use:

```bash
cd ui-web-client
npm run setup
npm start
```

It consists of:

1. Ruby on Rails backend API
2. React UI - based on the popular react-boilerplate
